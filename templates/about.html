📈 Technical Analysis CORE

Phân tích kỹ thuật cổ điển dựa trên 4 indicators:

  • RSI (14) — Relative Strength Index. Overbought >70, Oversold <30
  • MACD (12/26/9) — Moving Average Convergence Divergence. Crossover + histogram
  • EMA (21/50) — Exponential Moving Average crossover cho swing trading
  • Bollinger Bands (20, 2σ) — Volatility bands, mean reversion
  • Volume Analysis — Volume ratio vs SMA20, xác nhận breakout

Score range: -1.0 → +1.0. Weighted: RSI 30%, MACD 25%, EMA 25%, BB 20%.

📰 News Sentiment NLP

Phân tích tâm lý thị trường từ tin tức crypto realtime:

  • Sources: CoinDesk RSS, CoinTelegraph RSS (free, không cần API key)
  • NLP Engine: TextBlob polarity + domain-specific keyword boost
  • 3-tier keywords: High impact (ETF, halving) → Medium (rally, crash) → Low (upgrade, delay)
  • Sorting: Top 10 articles sorted by absolute impact

Nếu không có NewsAPI key, tự động dùng RSS feeds. Score: -1.0 → +1.0.

🤖 ML/AI Prediction ENSEMBLE

Ensemble machine learning dự đoán trend BTC:

  • XGBoost (55%) — Gradient boosting cho tabular data. 25+ features, 3-class (UP/FLAT/DOWN)
  • LSTM (45%) — Long Short-Term Memory neural network, sequence length 20
  • Features: Returns, volatility, momentum, MA ratios, RSI, MACD, EMA, BB position, lag features
  • Training: 365 ngày data, TimeSeriesSplit cross-validation, predict 5 candles ahead

Model được lưu tự động và load lại khi restart. Train lại mỗi 12h.

💎 SMC / ICT Analysis ADVANCED

Smart Money Concepts + Inner Circle Trader methodology:

  • Order Blocks (OB) — Demand/Supply zones nơi institutional orders đặt. 2-candle displacement confirmation
  • Fair Value Gaps (FVG) — Imbalances chưa được fill, giá có xu hướng quay lại
  • Liquidity Zones — Equal Highs/Lows nơi stop loss tập trung
  • BOS & CHoCH — Break of Structure (continuation) vs Change of Character (reversal)
  • Premium/Discount — Fibonacci OTE (61.8-78.6%) entry zone
  • Liquidity Sweep — Stop hunt detection với volume confirmation
🧠 Expert Analysis WYCKOFF

Phân tích chuyên sâu như trader chuyên nghiệp:

  • Market Structure — HH/HL/LH/LL counting → Uptrend/Downtrend/Ranging
  • Wyckoff Phase — Accumulation → Markup → Distribution → Markdown
  • Divergence — RSI vs Price (Bullish/Bearish divergence)
  • Multi-timeframe — H4 + M15 confluence check
  • Support/Resistance — Swing-based S/R với cluster analysis
